Food Manufacture I

Food Manufacture model from Model Building in Mathematical
Programming by H.P. Williams. 

How to run this example?
  * In CPLEX Studio IDE, import the example "Food manufacture 1" 
right click on the run configuration "Basic Configuration" and select "Run this"
  * In the command line, execute "oplrun -p <Install_dir>\opl\examples\opl\models\FoodManufacture1"
