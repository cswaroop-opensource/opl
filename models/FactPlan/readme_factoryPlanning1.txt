Factory Planning I

Factory Planning model from Model Building in Mathematical
Programming by H.P. Williams. We want to find the optimal mix
of products to manufacture given production capacities and
marketing limitations. 

How to run this example?
  * In CPLEX Studio IDE, import the example "Factory Planning" 
right click on the run configuration "Basic Configuration" and select "Run this"
  * In the command line, execute "oplrun -p <Install_dir>\opl\examples\opl\models\FactPlan 'Basic Configuration'"
