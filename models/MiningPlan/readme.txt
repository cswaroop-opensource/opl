Mine Planning

Problem 7 from Model Building in Mathematical Programming by
H.P. Williams. Decide which mines should be operated each year and
how much they should produce.


How to run this example?
  * In CPLEX Studio IDE, import the example "Mine planning" 
right click on the run configuration "Basic Configuration" and select "Run this"
  * In the command line, execute "oplrun -p <Install_dir>\opl\examples\opl\models\MiningPlan"
