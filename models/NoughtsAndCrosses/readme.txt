Three-dimensional Noughts and Crosses

Problem 17 from Model Building in Mathematical Programming by H.P. Williams.

How to run this example?
  * In CPLEX Studio IDE, import the example "Noughts and crosses", 
right click on the run configuration "Basic Configuration" and select "Run this"
  * In the command line, execute "oplrun -p <Install_dir>\opl\examples\opl\models\NoughtsAndCrosses"
