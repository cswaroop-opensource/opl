Blending I

Food Manufacture 1 problem from Model Building in Mathematical
Programming by H.P. Williams. This is a multi-period blending problem.


How to run this example?
  * In CPLEX Studio IDE, import the example "Blending Multi-period," 
right-click on the run configuration "LP formulation" and select "Run this"
  * In the command line, execute "oplrun -p <Install_dir>\opl\examples\opl\models\BlendingMultiPeriod 'LP formulation'"
