Blending II

Food Manufacture 2 problem from Model Building in Mathematical
Programming by H.P. Williams. This adds conditions to the original
food manufacturing problem to transform it into a Mixed Integer
Programming problem.


How to run this example?
  * In CPLEX Studio IDE, import the example "Blending Multi-period," 
right click on the run configuration "MIP formulation" and select "Run this"
  * In the command line, execute "oplrun -p <Install_dir>\opl\examples\opl\models\BlendingMultiPeriod 'MIP formulation'"
