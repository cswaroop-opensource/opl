Economic Planning 

Economic Planning model from Model Building in Mathematical
Programming by H.P. Williams. This investigates different
possible growth patterns for the economy over the next years.
Try different objectives by setting obj1, obj2 and obj3. 

- obj1: maximize capacity in year 5 
- obj2: maximize output in years 4 and 5, ignore exogenous demand 
- obj3: maximize manpower, ignore manpower constraint 

 
How to run this example?
  * In CPLEX Studio IDE, import the example "Economic planning" 
right click on the run configuration "Basic Configuration" and select "Run this"
  * In the command line, execute "oplrun -p <Install_dir>\opl\examples\opl\models\EconPlanning"
