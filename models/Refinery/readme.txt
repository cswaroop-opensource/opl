Refinery Planning

Refinery optimization: problem 6 from Model Building in
Mathematical Programming by H.P. Williams.


How to run this example?
  * In CPLEX Studio IDE, import the example "Refinery planning", 
right-click on the run configuration "Basic Configuration" and select "Run this"
  * In the command line, execute "oplrun -p <Install_dir>\opl\examples\opl\models\Refinery"
